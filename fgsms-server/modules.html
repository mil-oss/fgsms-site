Project Modules

This project has declared the following modules:

Name Description
fgsms Database Resources All things that run server side
fgsms Service Level Agreement Processor The SLA processor handles rules and actions invocations based on service owner defined rules
fgsms Common WS Test Utilities All base test cases for web services and database access
fgsms Reporting Service Processes reports on demand
fgsms Asynchronous Report Generator Processes scheduled reports based on user selection and schedule
fgsms Automated Reporting Service Configures scheduled reporting
fgsms Data Access Service Data Access Service
fgsms Data Collector Service DCS, collects data obviously
fgsms Data Pruner Removes expired records from the database based on policy
fgsms Dependency Scanner Scans through the database looking for chained service invocations
fgsms Policy Configuration Service Handles changing configuration and service policies
fgsms Status Service Providers getters and setters for all monitored service's status info
fgsms HornetQ JMS Agent Agent for HornetQ via remove JMX
fgsms Qpid JMX Agent Apache Qpid/Redhat MRG agent via remote JMX
fgsms UDDI Publisher Publishes availability and metrics data to a UDDI v3 server, such as Apache jUDDI
fgsms Statistics Aggregator Aggregates performance metrics and statistics to increase performance.
fgsms Apache ServiceMix/ActiveMQ JMX Agent Uses remote JMX to monitor components on an Apache Service Mix/Active MQ server
fgsms Status (Bueller) Agent The Status Agent runs with a computer/operating system and monitors the system itself and any running process on it.
fgsms Service WAR All of the fgsms's services end up here
fgsms UI Common User interface functions
fgsms User Interface (Bootstrap) fgsms's user interface based on Twitter Bootstrap 2
fgsms Tomcat JDCP Encryption Connection Supports the encryption of jdbc database credentials for tomcat
fgsms Tomcat Packaging All things that run server side
fgsms Server Crash Alerter A utility that alerts fgsms administrators in case of fgsms server crash
fgsms Database Export Tool A console tool to export the database content into sql scripts
fgsms Recryptor Enables admins to change encryption keys on the database
